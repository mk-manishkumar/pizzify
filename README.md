# Pizzify

This is a simple shopping cart application built with HTML, CSS, and JavaScript. It allows users to add items to their cart, view the total amount, and place an order by providing their personal information.

[Live Link](https://pizzify.netlify.app/)

## Features
- Add items to the cart
- Calculate and display the total amount
- Proceed to the order section
- Fill in personal information (name, address, postal code, mobile number)
- Submit the order and display a confirmation message
- Exit the application and reload the page

## Installation
- Clone the repository or download the source code files.
- Open the project folder on your local machine.

## Usage
- Open the index.html file in a web browser.
- The shopping cart page will be displayed.
- Click the "Add to Cart" button next to the desired items to add them to the cart.
- The total amount will be automatically calculated and displayed.
- Click the "Order" button to proceed to the order section.
- Fill in the required personal information fields.
- Click the "Submit" button to place the order.
- A confirmation message will be displayed.
- To exit the application, click the "Exit" button.

## Customization

You can customize the application by modifying the HTML, CSS, and JavaScript code according to your requirements. Here are some possible customizations:

- Change the layout and styling of the shopping cart page.
- Add more items to the cart or modify the existing items.
- Customize the order form fields and validation.
- Enhance the user interface with additional features or animations.
